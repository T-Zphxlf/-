# 轻语输入法 (VoiceType)

<p align="center">
  <strong>为 Android TV 打造的轻量语音输入法</strong>
</p>

## ✨ 特性

- 🎤 **语音优先**：一键语音输入，再按结束，支持方言、流行词、影视剧名识别
- ⌨️ **26键+9宫格**：电视遥控器友好的键盘布局，D-Pad 导航选字
- 📺 **TV 适配**：专为 Android TV 遥控器优化，大字体、高对比度
- 🔤 **词库更新**：支持在线更新流行词、影视剧名词库
- 🪶 **极致轻量**：仅 arm64-v8a 架构，Android 9+ 即可运行

## 🔧 编译

本项目使用 GitHub Actions 自动编译，也可本地编译：

```bash
./gradlew assembleRelease
```

## 📋 系统要求

- Android 9.0 (API 28) 及以上
- arm64-v8a 架构
- 麦克风权限（语音输入）
